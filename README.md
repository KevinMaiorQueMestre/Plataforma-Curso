# Template Antigravity Base (Arquitetura DOE)

Este é o template padrão para a inicialização de projetos limpos, como plataformas educacionais, sistemas de ouvidoria e dashboards. Ele segue a Arquitetura DOE (Directive, Orchestration, Execution) para minimizar alucinações da IA.

## Estrutura
- `/directives`: Contém as regras de negócio e manuais de instrução em Markdown (Camada 1).
- `/execution`: Contém scripts determinísticos e funções de backend (Camada 3).
- `.tmp/`: Arquivos temporários e descartáveis.
- `.env`: Variáveis de ambiente e chaves de API (protegidas).
